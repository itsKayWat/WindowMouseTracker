RelativeMouseCoords

A precise mouse coordinate tracking tool for window-relative positions

Purpose

RelativeMouseCoords helps developers and users track mouse coordinates relative to the current window position, essential for UI automation and testing tasks.

Features

  • Real-time coordinate tracking
  • Window-relative positioning
  • Dark mode interface
  • Hotkey toggle (Alt+')
  • Minimalistic floating display
  • System tray integration

Installation

Prerequisites:

  • Python 3.6 or higher
  • Windows Operating System

Steps:

  1. Clone the repository
  2. Run requirements.py
  3. Launch mouse_tracker.py

Usage

  1. Run the application
  2. Use Alt+' to toggle coordinate display
  3. Hover over any window element to see coordinates
  4. Coordinates are relative to the current window

Use Cases

  • UI Automation Development
  • Application Testing
  • UI Element Mapping
  • Layout Debugging

Example Scenario

A QA engineer needs to create automated tests for a desktop application. They use RelativeMouseCoords to:

  1. Identify exact button positions
  2. Map form field locations
  3. Document menu item coordinates
  4. Create precise automation scripts
